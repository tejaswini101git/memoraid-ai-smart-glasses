# Hardware Components

The MEMORAID smart assistive system is built using the following hardware components:

![Hardware Setup](memoraid_prototype.jpg)

## Core Components

- ESP32-CAM module (camera + processing)
- OV2640 Camera Sensor (built-in)
- Bone Conduction Speaker
- Audio Amplifier Module

- Breadboard (for prototyping)
- Jumper Wires

## Power System

- Li-ion Battery
- Voltage regulator / USB supply

## Connectivity

- Wi-Fi (ESP32 built-in)

This hardware setup enables real-time image capture, processing, and audio feedback for cognitive assistance.
