# Software Overview

This directory will contain the software components of the MEMORAID system.

## Planned Modules

- Face detection and recognition algorithms
- Embedded code for ESP32-CAM
- Audio processing and playback logic
- Mobile application for caregiver notifications

## Technologies

- Python (AI/ML models)
- OpenCV (image processing)
- ESP32 firmware (C/C++)
- Mobile App (future integration)
